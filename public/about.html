About this App

Tools used

  • Phaser JS provides the client-side game engine

  • The Server is based off of Node JS, Express, and WebSockets

  • SQLite3 is used to store user data

  • Vue JS is used as a client-side framework

  • Bootstrap is an open source toolkit for HTML, CSS, an JS.
